# architecture-fe 

前端工程化脚手架

# command

### node_modules
	```bash
	npm install
	```

### dev
	```bash
	    npm run dev
	    open localhost:7778
	```
### prod
	```bash
	    npm run prod
	```

# main functionalities

	- npm scripts [dev product]

	- babel [ES6 ES5]

	- webpack [img, css, js]

	- jest [单元测试]

	- react 框架适配
